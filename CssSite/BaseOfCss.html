HTML / CSS Hyper Text MarkUp Language - Cascading Style Sheets
HTML 이란? 월드 와이드웹 문서를 작성하는 MarkUp Language이다. 여러 태그들로 구성되어 있으며, 각 태그들을 사용하여 원하는 형태의 문서를 만들어 갈 수 있다. Hyper Text의 원리를 이용하여 다양한 정보를 손쉽게 검색하여 볼 수 있게 해주는 언어이다.
자주 사용되는 HTML 태그들 div span a ul/ol li img h1~h6 header section footer
CSS란? HTML, XHTML, XML과 같은 문서의 스타일을 꾸밀 때 사용하는 스타일 시트 언어이다. 글꼴이나, 배경색, 너비와 높이, 위치 등을 지정하거나 웹 브라우저, 스크린 크기, 장치에 따라서 화면을 다르게 표시될 수 있도록 지정한 것이 큰 특징이라고 할 수 있다.
